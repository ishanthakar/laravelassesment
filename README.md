## Project setup steps
- Create .env file
- Copy .env.example to .env

- Run following commands
1) create table
php artisan migrate

2) Create Admin user
php artisan db:seed

- Import following postman collection for use apis after installing leatest postman collection as we are adding bearer token to use auth apis.
